# 🌌 Antigravity Orchestrator

The **Antigravity Orchestrator** is the meta-layer that drives high-end, autonomous code generation. It uses a "Skill Brain" registry to inject expert logic and design sensibilities into every task.

## ⚡ Mandatory Workflow

The orchestrator follows a strict 3-step protocol:
1.  **Analysis**: Determining the scale, stack, and aesthetic of the task.
2.  **Skill Brain Match**: Identifying the best experts (Design, Logic, Motion) to handle the implementation.
3.  **Auto-Action**: Generating a stack proposal and initializing code generation immediately.

## 🛡️ Autonomous Rules

- **Continuous Flow**: No hard stops or unnecessary permission requests.
- **Skill Injection**: Actively pulls rules from the matched skills to ensure "Anti-Slop" quality.
- **Premium Defaults**: Always aims for Apple-inspired, minimalist, and motion-rich results.
