---
description: Create or run automated workflow
---

Use workflow-composer to:

CREATE MODE:
- Build YAML workflow with multiple skills
- Add conditional logic (IF/THEN/ELSE)
- Configure parallel execution
- Set up error handling

RUN MODE:
- Execute existing workflow
- Track progress in real-time
- Generate execution report

Example: "Create workflow that runs tests, security scan, then deploys if all pass"
