# ✨ Taste Skill: The Visual Intelligence Registry

**Taste Skill** is a curated collection of specialized design skills designed to enforce high-end visual aesthetics and professional UI/UX standards.

## 🗂️ Sub-Skills & Capabilities

- **[Minimalist UI](./minimalist-ui)**: Clean, Apple-inspired, and focused.
- **[Emil Design Engineering](./emil-design-eng)**: Motion physics and UI polish.
- **[Brandkit](./brandkit)**: Visual identity and asset management.
- **[Industrial Brutalist](./industrial-brutalist-ui)**: High-contrast, bold, and functional design.
- **[Image to Code](./image-to-code)**: Visual-to-functional transformation logic.

## 🎨 Core Objectives

- **Anti-Slop Enforcement**: Eliminating generic, low-quality UI patterns.
- **Visual Cohesion**: Ensuring all components share a consistent design language.
- **Motion Mastery**: Integrating sophisticated animations that feel natural and responsive.
